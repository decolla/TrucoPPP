# Uno-Truco-game-Java 🃏🎴

Este projeto é uma implementação em **Java** que reúne dois dos jogos de cartas mais populares: **Uno** e **Truco**. Desenvolvido com foco em lógica de programação orientada a objetos (POO), o projeto permite aos utilizadores jogarem partidas rápidas e interativas.

## 🚀 Sobre o Projeto

O **Uno-Truco-game-Java** foi criado para demonstrar a aplicação de conceitos fundamentais de Java, como herança, polimorfismo, coleções e interfaces, aplicados ao domínio de jogos de mesa.

### Funcionalidades

  - 🃏 **Modo Uno:** Regras clássicas, cartas especiais (Inverter, Saltar, +2, +4) e escolha de cores.
  - 🎴 **Modo Truco:** Sistema de tentos, manilhas e a opção de "pedir truco" (6, 9, 12).
  - 👤 **Interface:** [Consola / GUI - *escolher a que se aplica ao teu caso*].
  - 🔄 **Baralhos Dinâmicos:** Gestão eficiente da pilha de compra e descarte.

## 🛠️ Tecnologias Utilizadas

  * **Linguagem:** Java 17+ (ou a versão que estiveres a usar)
  * **Paradigma:** Orientação a Objetos
  * **Build System:** [Maven / Gradle / Nenhum]

## 📋 Pré-requisitos

Antes de começares, certifica-se de que tens instalado:

  * [Java JDK](https://www.oracle.com/java/technologies/downloads/) (versão 11 ou superior recomendada)
  * Uma IDE (IntelliJ IDEA, Eclipse ou VS Code) ou terminal.

## ⚙️ Como Executar

1.  **Clona o repositório:**

    ```bash
    git clone https://github.com/decolla/Uno-Truco-game-Java.git
    ```

2.  **Entra na pasta do projeto:**

    ```bash
    cd Uno-Truco-game-Java
    ```

3.  **Compila o projeto:**

    ```bash
    javac -d bin src/*.java
    ```

4.  **Executa o jogo:**

    ```bash
    java -cp bin Main
    ```
